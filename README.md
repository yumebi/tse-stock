# TSE Stock — 東証リアルタイム株価アプリ

Tauri v2 + Rust で構築された、東京証券取引所のリアルタイム株価モニタリングアプリです。

![dark-theme](https://img.shields.io/badge/theme-dark-0d1117) ![rust](https://img.shields.io/badge/rust-1.95-orange) ![tauri](https://img.shields.io/badge/tauri-v2-blue)

## 機能

### 📊 リアルタイム株価
- 銘柄コード（4桁）で東証銘柄を追加・監視
- 現在株価・前日比・始値・高値・安値・出来高を表示
- 高値・安値には**時刻**も表示（分足データから抽出）
- 30秒間隔で自動更新

### 📈 テクニカル指標
| 指標 | 内容 |
|---|---|
| MA5 / MA25 / MA75 | 単純移動平均線（5日・25日・75日） |
| MACD | MACD(12,26) + シグナル(9) |
| RSI | 14日RSI |

### 🚨 シグナル検知
| カテゴリ | シグナル |
|---|---|
| **MAクロス** | ゴールデンクロス(GC) / デッドクロス(DC) — MA5×MA25, MA25×MA75 |
| **MACD** | シグナル線クロス（買い/売り）+ 現在状態（買い継続/売り継続） |
| **RSI** | 70以上で買われすぎ / 30以下で売られすぎ |
| **ボリンジャーバンド** | ±2σタッチ |
| **急変** | 前日比±3%（大幅高/安）・±5%（急騰/急落） |
| **出来高** | 平均の1.5倍以上でスパイク検知 |
| **日中位置** | 高値圏 / 安値圏 |
| **ATR** | 14日ATRの2倍超で高ボラティリティ |
| **連続方向** | 3日以上の続騰/続落 |

### 🗂 日本語企業名
- 約200銘柄の日本語名を内蔵マッピング
- マッピングにない銘柄は [みんかぶ](https://minkabu.jp) から自動取得
- 取得済みの名前はローカルにキャッシュ

### 🔧 その他
- ▲▼ボタンで銘柄の並べ替え
- 列ヘッダークリックでソート（昇順/降順）
- 銘柄リストは自動保存（再起動後も復元）
- ダークテーマ
- 右下に凡例（マウスオーバーで詳細）

## 技術スタック

| 層 | 技術 |
|---|---|
| フレームワーク | [Tauri v2](https://v2.tauri.app/) |
| バックエンド | Rust (reqwest, serde, chrono, tokio) |
| フロントエンド | Vanilla JS + CSS Grid |
| バンドラ | Vite |
| データソース | [Yahoo Finance API](https://query1.finance.yahoo.com/v8/finance/chart/) + [みんかぶ](https://minkabu.jp) |

## 開発環境のセットアップ

### 必要条件
- [Rust](https://rustup.rs/) 1.70+
- [Node.js](https://nodejs.org/) 18+
- Windows: [WebView2](https://developer.microsoft.com/microsoft-edge/webview2/)（通常はOSに組み込み済み）

### インストール

```bash
# クローン
git clone https://github.com/yumebi/tse-stock.git
cd tse-stock

# 依存パッケージのインストール
npm install

# 開発モードで起動（ホットリロード）
npm run tauri dev

# ビルド
npm run tauri build
```

ビルド後の実行ファイルは `src-tauri/target/release/tse-stock.exe` に生成されます。
